Privacy Policy

Effective date: 2026-04-23

Manual snapshots happen only when the user clicks Snapshot now. Automatic snapshots are off by default and start only after the user clicks Enable autosave.

Data handled

Session Rescue stores tab URLs, tab titles, window grouping, pinned and active tab state, and capture timestamps. It does not read page content, form fields, cookies, passwords, or account data.

Storage and network behavior

Snapshots are stored locally in IndexedDB in the user's Chrome profile. Session Rescue does not send session snapshots, tab URLs, searches, imports, exports, or usage events to a server.

Export, import, and deletion

Users can export JSON backups, import compatible JSON backups, delete individual snapshots, or clear all snapshots from the library. Imported backups are limited by file size, tab count, and restorable URL schemes.

Chrome Web Store User Data Policy

Use of information received from Chrome APIs complies with the Chrome Web Store User Data Policy, including the Limited Use requirements. Session data is used only for local snapshot, restore, search, export, and import features.

No ads or sale

Session Rescue has no ads, analytics, sale of user data, or third-party transfer of user data.
